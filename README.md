# Web Traffic Analytics Project

## Overview
This project analyzes website traffic data to understand user behavior and improve website performance. It helps identify key metrics such as page views, session duration, and drop-off rate using Python and Streamlit.

---

## Objectives
- Analyze user behavior on a website
- Track page visits and session duration
- Identify drop-off points
- Visualize data using charts
- Build an interactive dashboard

---

## Project Structure

analysis.py,

README.md,

---

## Dataset Description
- user_id → Unique user identifier  
- page → Page visited  
- session_time → Time spent on page (in seconds)  
- clicked → Whether user clicked (Yes/No)  
- exit → Whether user exited the site  

---

## Tools Used
- Python
- Pandas
- Streamlit
- Matplotlib (optional)

---

## How to Run

### Install dependencies
pip install -r requirements.txt

### Run analysis
python analysis.py

### Run dashboard
streamlit run app.py

---

## Key Insights
- Home page has highest traffic
- Average session duration calculated from user behavior
- Drop-off rate shows user exit percentage
- Checkout page needs improvement
